Usage

<cron-generator ng-model="cronExpression" config="cronOptions"></cron-generator>
angular.module('ExampleApp', ['angular-cron-generator'])
  .controller('ExampleCtrl', ['$scope', function ($scope) {
    $scope.cronOptions = {
      allowMultiple: true,
      quartz: true
    };
  }]);

Options

Options Description Type Default
allowMultiple Allows you to select multiple values boolean defaults to false
quartz Allows you to get any of Unix and quartz format output boolean defaults to false
